# 🚀 App-music Backend API

Backend escalable para la gestión de rankings de eventos musicales integrando la API de Spotify.

## 🛠️ Tecnologías
- **Node.js / Express** (Servidor) ✅
- **MongoDB Atlas** (Base de datos NoSQL)❌
- **JWT & Bcrypt** (Seguridad y Auth)❌
- **Render** (Despliegue)❌

## ⚙️ Instalación Local Basica para ver el funcionamiento del server

1. Clona el repositorio:
   ```bash
   git clone https://github.com ...
   ```

2. Instala las dependencias

    ```bash 
    npm install 
    ```
3. Configura las Variables de Entorno:
    - Crea un archivo .env basado en el siguiente esquema:
    ```bash 
    PORT="el puerto que tu prefieras o este disponible"
    ```
4. Ejecuta el servidor en modo desarrollo:
    ```bash
    npm run dev
    ```
